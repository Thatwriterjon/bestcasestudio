# Best Case Studio — Website

Marketing site for [Best Case Studio](https://bestcasestud.io), a B2B SaaS case study production service.

Built with [Astro](https://astro.build) and Tailwind CSS. Deployed on Cloudflare Pages.

## Pages

- `/` — Homepage (hero, services, case studies, team, testimonials, referral programme, FAQ)
- `/pricing` — Pricing packages
- `/contact` — Calendly booking embed + contact form
- `/articles` — Blog (coming soon)

## Commands

| Command           | Action                                      |
| :---------------- | :------------------------------------------ |
| `npm install`     | Install dependencies                        |
| `npm run dev`     | Start local dev server at `localhost:4321`  |
| `npm run build`   | Build production site to `./dist/`          |
| `npm run preview` | Preview production build locally            |
